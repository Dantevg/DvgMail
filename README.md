# DvgMail
### How to install
#### Client
*Installation for client is coming soon...*

#### Server
1. Run in the console: `> pastebin get F9TJmRQR`<br>
2. create a startup file with the following code and change `<side>` to your wireless modem side.

    ```lua
    rednet.open("<side>")
    shell.run("dvgmailserver")
    ```

### Requirements
* Mail server
* dvg API
* cctf API
* advanced computer
